# GAAH Research Dashboard

Professional Streamlit workspace for analysing phase transitions and dynamical regimes of the generalized Aubry–André–Harper (GAAH) model.

## Features
- Modular multi-page layout (spectral, dynamics, data explorer, mobility-edge, fractal analysis).
- Plotly-based visuals with dark theme, responsive layout, and download buttons (PNG + JSON).
- Sidebar filters for `L`, `λ`, `φ`, and boundary conditions with live dataset counts.
- Scientific interpretation helpers: ⟨r⟩ phase tags, MSD trend diagnosis, mobility-edge thresholding.
- Fractal-dimension proxy (D₂) from IPR statistics with configurable dataset cap.

## Physics & Model Details

The dashboard analyses the **Generalized Aubry–André–Harper (GAAH)** model (Ganeshan et al., PRL 2015).

### Hamiltonian
$$ H = -t \sum_{n} (|n\rangle\langle n+1| + \text{h.c.}) + \sum_{n} V_n |n\rangle\langle n| $$

### Onsite Potential
$$ V_n = 2\lambda \frac{\cos(2\pi b n + \phi)}{1 - \alpha \cos(2\pi b n + \phi)} $$

### Terms
- **$t$**: Hopping amplitude (kinetic energy).
- **$\lambda$**: Potential strength.
- **$\alpha$**: Deformation parameter ($\alpha \in (-1, 1)$). $\alpha=0$ recovers the standard AA model.
- **$b$**: Irrational frequency (e.g., golden mean).
- **$\phi$**: Phase offset.

### Mobility Edge
The exact boundary between localized and extended states is given by:
$$ \alpha E = 2 \text{sgn}(\lambda) (|t| - |\lambda|) $$

## Numerical Implementation

The underlying simulation (`gaahmodel_parallel_timeevol.py`) employs:

1.  **Sparse Hamiltonian Construction**: Efficient CSR matrix format for large $L$.
2.  **Hybrid Diagonalization**:
    *   `scipy.sparse.linalg.eigsh` for spectral subsets (large systems).
    *   `scipy.linalg.eigh` for full spectrum (small systems).
3.  **Crank–Nicolson Time Evolution**:
    *   Implicit, unitary, and unconditionally stable integration.
    *   Tracks **MSD** ($\sigma^2(t)$) and **Survival Probability** ($P(t)$).
4.  **Parallelization**: `joblib`-based distribution of $(\lambda, L, \phi)$ tasks.

## Getting started
1. (Optional) create and activate a virtual environment.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Copy `.npz` results from `gaahmodel_parallel_timeevol.py` (or `full_pipeline_gaah.py`) into `gaah-dashboard/data/`.
4. Launch Streamlit:
   ```bash
   streamlit run app.py
   ```
5. Use the sidebar to filter datasets; navigate between pages via the Streamlit page selector.

> **Tip:** Eigenstate-based pages require eigenvectors stored as `V` in the `.npz` files. Time-evolution plots require simulations run with `--evolve`.

## Scientific cues baked in
- ⟨r⟩ ≈ 0.386 ⇒ localized (Poisson). ⟨r⟩ ≈ 0.530 ⇒ extended (GOE). Intermediate values flag criticality.
- MSD saturation prompts a localization warning; continued growth signals extended transport.
- Mobility-edge page highlights energy-dependent transitions with an adjustable IPR threshold.
- Fractal page estimates D₂ ≈ −log(IPR)/log(L) as a quick proxy for multifractality.

## Extensibility
- Add new pages under `pages/` following the existing component pattern.
- Extend `utils.py` with additional observables (e.g., entanglement, Floquet analysis).
- `components/plot_helpers.py` centralises export behaviour; customise to add SVG or PDF as needed.
